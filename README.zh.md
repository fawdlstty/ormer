### 各框架特点总结

#### Ormer 优势场景 ✅
- **快速原型开发**：模型定义一次即可，自动生成建表 SQL，无需额外工具
- **学习成本低**：直观的 LINQ 风格 API，1 天即可上手使用
- **代码简洁**：无代码重复，无需维护多份定义或迁移文件（基础场景）
- **编译期安全**：完整的类型推导，错误在编译期即可发现
- **轻量级项目**：包体积小，编译速度快，适合微服务和小型项目
- **多数据库切换**：统一 API，切换数据库无需修改业务代码
